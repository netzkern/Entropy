# netzkern UserScripts

UserScripts sind kleine, durch eine Browser-Extension ausgeführte, JavaScripts, die Websites nach Belieben anpassen, erweitern oder reparieren können.

## Wie benutze ich UserScripts?

Die UserScripts können in Firefox mit GreaseMonkey und in Chrome mit TamperMonkey verwendet werden.
